To Start/View Project Please Use The Following Instructions -

1. install dependencies for backend server - (in mern-backend folder) :
	a.) npm init -y 
	b.) npm install express mongoose cors
	c.) start backend - node server.js

2.) I've used Thunder Client for adding data on server using "Post" method.

3.) To start frontend do this in (mern-frotend folder ) - 
	a.) npx create-react-app mern-frontend.
	b) npm start -> to start frontend.
